# Virtual Stock Market Simulator

A web-based virtual stock trading simulator where users can buy and sell stocks with dynamic price movement and portfolio tracking.

## Features
- Live stock price simulation
- Buy and sell system
- Portfolio balance tracking
- Net profit calculation
- Dark mode interface
- Local data saving
- Interactive graph display

## How to Use
1. Open the live site
2. Choose a stock
3. Enter quantity
4. Buy or sell
5. Watch your portfolio grow or fall

## Future Plans
- News-based price events
- Smarter AI market behavior
- Better UI and animations
- Advanced portfolio analytics
